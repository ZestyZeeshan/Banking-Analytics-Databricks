**Banking Analytics Platform using Databricks \& PySpark**



=> **Project Overview**


This project demonstrates an end-to-end Banking Analytics Data Engineering pipeline built using Databricks Free Edition and PySpark following Medallion Architecture.



=> **Tech Stack**



\- Databricks

\- PySpark

\- Delta Lake

\- SQL

\- Medallion Architecture



=> **Medallion Architecture**



Bronze → Silver → Gold



=> **Features**



\- Data Ingestion

\- Bronze Layer

\- Silver Layer Data Validation

\- Referential Integrity Checks

\- Rejected Records Tracking

\- Gold Layer KPIs

\- Fraud Detection

\- Dashboard Visualizations





=> **Key Highlights(Topics) Covered in this project**



\- PySpark Transformations

\- Delta Lake

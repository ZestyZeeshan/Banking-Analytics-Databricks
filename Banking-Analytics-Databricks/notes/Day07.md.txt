New PySpark Concepts Learned
withColumn()

Purpose:

Create a new column.

Example:

.withColumn("risk_category", ...)
lit()

Purpose:

Add a constant value to every row.

Example:

lit("High Value Transaction")

Output:

fraud_reason = High Value Transaction
when()

Purpose:

Apply conditional logic.

Example:

when(amount > 50000, "High Risk")

Equivalent SQL:

CASE WHEN amount > 50000
THEN 'High Risk'
END
otherwise()

Purpose:

Default condition when no rule matches.

Example:

.otherwise("Low Risk")

Q1. What is Fraud Detection?

Answer:

Fraud Detection is the process of identifying suspicious transactions based on predefined business rules or machine learning models.

Q2. How did you implement fraud detection in your project?

Answer:

I implemented a rule-based fraud detection framework in Databricks using PySpark. Transactions above ₹50,000 were flagged as suspicious and stored in a fraud alerts table.

Q3. What was your fraud rule?

Answer:

Transactions with an amount greater than ₹50,000 were classified as High Value Transactions and flagged for investigation.

Q4. What is risk categorization?

Answer:

Risk categorization classifies transactions into Low, Medium, and High Risk based on transaction amount.

Q5. What is the purpose of the fraud_alerts table?

Answer:

The fraud_alerts table stores suspicious transactions separately for monitoring, auditing, and investigation.

Q6. What does withColumn() do?

Answer:

withColumn() is used to create or modify a column in a DataFrame.

Q7. What does lit() do?

Answer:

lit() creates a constant value column for all rows in a DataFrame.

Q8. What does when() do?

Answer:

when() is used for conditional logic in PySpark and works similarly to SQL CASE WHEN statements.

Q9. What KPIs did you create for fraud monitoring?

Answer:

I created Total Transactions, Total Fraud Alerts, and Fraud Percentage KPIs for fraud monitoring and reporting.

Q10. Why is fraud detection important in banking?

Answer:

Fraud detection helps banks identify suspicious transactions early, reduce financial losses, and improve customer security.
